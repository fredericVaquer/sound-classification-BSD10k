# Sound Classification & Audio-Language Retrieval (BSD10k)

Assignment work for the *Advanced Computing Techniques for Sound and Music* module (SMC, UPF 2025–26).  
Explores heterogeneous sound classification using the real [BSD10k dataset](https://github.com/allholy/BSD10k)
and text-based audio retrieval via CLAP embeddings.

Reference paper: Anastasopoulou et al., ["Heterogeneous sound classification with the Broad Sound Taxonomy and Dataset"](https://dcase.community/documents/workshop2024/proceedings/DCASE2024Workshop_Anastasopoulou_39.pdf), DCASE 2024.

## Data

Unzip so the repo root contains `audio/` and `metadata/` as shown above.

### Metadata schema (`BSD10k_metadata.csv`)

| Column | Description |
|---|---|
| `sound_id` | Freesound ID; audio file is `audio/{sound_id}.wav` |
| `class` | Second-level class code (e.g. `MUS`, `SPE`) |
| `class_idx` | Integer index 0–22 |
| `class_top` | Top-level class code |
| `title` | Sound title from Freesound |
| `tags` | Tags from Freesound |
| `uploader` | Freesound uploader |
| `license` | License URL |

Class names are in `BST_description.csv`.  
The taxonomy has **5 top-level** and **23 second-level** classes.

## Workflow

Run the notebooks in order:

### 1 · `1_subset_and_embeddings.ipynb`
- Loads and inspects the full BSD10k taxonomy
- Selects a subset of classes (edit `SELECTED_CLASSES`)
- Saves `metadata/subset_metadata.csv`

### 2 · `2_sound_classification_top_classes.ipynb`
- predefined train/test split in BSD10k
- KNN classification with StandardScaler
- Accuracy comparison across embedding types
- Confusion matrix, per-class F1, misclassification listening

### 2 · `2_sound_classification_sub_classes.ipynb`
- predefined train/test split in BSD10k
- KNN classification with StandardScaler
- Accuracy comparison across embedding types
- Confusion matrix, per-class F1, misclassification listening
- Classification of Freesound sounds with the k-NN CLAP embeddings classifier

### 3 · `3_text_audio_retrieval.ipynb`
- CLAP nearest-neighbour text-to-audio retrieval
- Prompt sensitivity experiments with Jaccard overlap analysis
- Precision@5 evaluation per query category
